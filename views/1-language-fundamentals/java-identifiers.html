Java Identifiers
This is third header

A name in java programming is called identifiers which can be used for identitication purpose.

It can be method name, variable name, class name or label name.

e.g. 
  
    class Test{
      public static void main(String[] args){
        int x = 10; 
      }
    }

In above example, there are 5 identifiers
Test - class name
main - method name
String - pre-defined class name
args - variable name of type string
x - variable name of type int
